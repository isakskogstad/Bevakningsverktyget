Om detta verktyg

Detta verktyg anvander ditt Ratsit-foretagskonto for att hamta inkomstdeklarationer. Kostnader debiteras automatiskt enligt din betaltjanst.

  • Forvarsinkomst: Lon, pension, sjukpenning etc.
  • Kapitalinkomst: Ranta, utdelning, kapitalvinster
  • Slutlig skatt: Total skatt efter avdrag

OBS: Detta verktyg kräver att scrapern körs lokalt via CLI. Webbgränssnittet är endast för demonstration. Se CLI-instruktioner nedan.

CLI-anvaendning

# Installera dependencies $ cd lib/scrapers && npm install # Konfigurera miljovariabler $ export RATSIT_EMAIL="ditt@foretag.se" $ export RATSIT_PASSWORD="ditt-losenord" # Sok efter person $ node ratsit-income-scraper.js search "Anna Andersson" $ node ratsit-income-scraper.js search "Erik Eriksson" --location=Stockholm # Hamta inkomstuppgifter $ node ratsit-income-scraper.js income "Isak Skogstad" $ node ratsit-income-scraper.js income "Anna Andersson" --birth-year=1985 --location=Goteborg # Hamta fran specifik profil $ node ratsit-income-scraper.js profile "https://www.ratsit.se/person/..." # Visa browser (for debug/CAPTCHA) $ node ratsit-income-scraper.js income "Test" --visible
