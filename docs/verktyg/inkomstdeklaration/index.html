CLI-verktyget

For att hamta nya inkomstuppgifter fran Ratsit.se maste du kora CLI-verktyget lokalt. Data sparas automatiskt till databasen med --save flaggan.

Installation och anvandning

# Installera dependencies $ npm install # Konfigurera miljovariabler $ export RATSIT_EMAIL="ditt@foretag.se" $ export RATSIT_PASSWORD="ditt-losenord" $ export SUPABASE_URL="https://xxx.supabase.co" $ export SUPABASE_KEY="din-api-nyckel" # Sok efter person $ node lib/scrapers/ratsit-income-scraper.js search "Anna Andersson" # Hamta inkomst och spara till Supabase $ node lib/scrapers/ratsit-income-scraper.js income "Isak Skogstad" --save $ node lib/scrapers/ratsit-income-scraper.js income "Anna" --birth-year=1985 --save # Hamta fran specifik profil och spara $ node lib/scrapers/ratsit-income-scraper.js profile "https://..." --save # Visa browser (for debug/CAPTCHA) $ node lib/scrapers/ratsit-income-scraper.js income "Test" --visible --save
